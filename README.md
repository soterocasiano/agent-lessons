# agent-lessons

Repository to hold lessons and teaching documents generated by Agents. Lessons are written to teach the reader a topic directly — the agent acts as the teacher, not as the author of a plan for someone else to present.

## Directory Structure

| Directory | Purpose |
|-----------|---------|
| [`lessons/`](lessons/) | Structured lessons that teach the reader directly |
| [`lessons/impromptu/`](lessons/impromptu/) | Informal, on-the-fly lessons from agent interactions |
| [`skills/`](skills/) | Agent skill definitions and implementations |
| [`docs/`](docs/) | Subsidiary documentation (guides, glossaries, reference material) |
| [`scripts/`](scripts/) | Automation scripts, e.g. generating lessons with the GitHub Copilot SDK |
