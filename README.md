# agent-lessons

Repository to hold lesson plans and teaching documents generated by Agents.

## Directory Structure

| Directory | Purpose |
|-----------|---------|
| [`lessons/`](lessons/) | Structured lesson plans |
| [`lessons/impromptu/`](lessons/impromptu/) | Informal, on-the-fly lessons from agent interactions |
| [`skills/`](skills/) | Agent skill definitions and implementations |
| [`docs/`](docs/) | Subsidiary documentation (guides, glossaries, reference material) |
| [`scripts/`](scripts/) | Automation scripts, e.g. generating lesson plans with the GitHub Copilot SDK |
