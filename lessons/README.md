# Lessons

This directory contains structured lessons generated by agents. Each lesson teaches the reader a topic directly — written as the actual teaching content, not a plan for a separate presenter.

## Structure

- **`impromptu/`** – Informal, on-the-fly lessons that arise during agent interactions.

## Adding a Lesson

Create a new Markdown file (e.g., `lesson-<topic>.md`) in this directory. Include a title, learning objectives, the full teaching content (explanations, examples, checks for understanding), and any follow-up exercises.
