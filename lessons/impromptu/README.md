# Impromptu Lessons

This directory contains informal, on-the-fly lessons that emerge spontaneously during agent interactions or teaching sessions.

Unlike structured lesson plans, impromptu lessons may be shorter, less formal, and driven by immediate questions or discoveries.

## Adding an Impromptu Lesson

Create a new Markdown file named after the topic (e.g., `<topic>.md`) and capture the key insights, examples, and takeaways.
